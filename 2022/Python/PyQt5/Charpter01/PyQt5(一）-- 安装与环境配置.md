## 1 PyQt5 图形界面开发工具

Qt 库是跨平台的 C++ 库的集合，是最强大的 GUI 库之一，可以实现高级 API 来访问桌面和移动系统的各种服务。
PyQt5 是一套 Python 绑定 Digia QT5 应用的框架。PyQt5 实现了一个 Python模块集，有 620 个类，6000 个函数和方法。
PyQt5 的优点：简单好用，功能强大， 跨平台支持，文档齐全，稳定性高，生态支持，开源免费。
